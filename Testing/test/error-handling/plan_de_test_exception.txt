Plan de Test pour les Classes d'Erreur
Objectif
Vérifier que chaque classe d'erreur est correctement instanciée avec le message d'erreur approprié et qu'elle hérite de la classe Error.

1. Classe InvalidEventNameError
Cas de Test:

Cas de test 1.1: Instancier InvalidEventNameError avec un message d'erreur.
Entrée: error = 'Invalid event name'
Sortie attendue: L'instance doit avoir error.message égal à 'Invalid event name' et être une instance de Error.
